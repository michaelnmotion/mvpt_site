# MVPT Site (Eleventy)

## Install
```bash
npm install
```

## Run locally
```bash
npm run dev
```

## Build
```bash
npm run build
```

Output is in `dist/`.

## Edit once, update everywhere
- Header: `src/_includes/partials/header.njk`
- Footer: `src/_includes/partials/footer.njk`
- Promo bar: `src/_data/promo.json` (and home page flag in `src/pages/index.njk`)
- Pages: `src/pages/*.njk`
- Assets (CSS/JS/images/video): `src/assets/` (copied to site root)
